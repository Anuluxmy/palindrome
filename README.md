# Palindrome

Aim
-------
```
To write a function to calculate the largest palindromic number from the product of two 3-digit numbers (100 to 999).
```
Clone repository
----------
```
* git clone https://github.com/Anuluxmy/palindrome.git
* cd palindrome
```

To run Palindrome
---------
```
* bundle install
* cd lib
* ruby palindrome.rb
```

To run test
----
```
* rspec
```
